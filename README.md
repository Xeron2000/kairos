# kairos

> 古希腊语：关键时刻，恰当时机

Crypto trading agent system for Hermes Agent. Based on Bit浪浪's trading philosophy.

## Commands

```bash
kairos cycle                    # Market cycle
kairos scan                     # Scan symbols
kairos box --symbol BTC/USDT    # Box pattern
kairos signal --symbol BTC/USDT # Trading signal
kairos pos                      # Positions
kairos risk                     # Risk status
kairos history                  # Trade history
```

## Skills

| Skill | Description |
|-------|-------------|
| `kairos-cycle` | 春夏秋冬周期判断 |
| `kairos-scanner` | 量化选币 |
| `kairos-box` | 箱体识别 |
| `kairos-signal` | 交易信号 |
| `kairos-position` | 仓位管理 |
| `kairos-risk` | 风险控制 |
| `kairos-review` | 复盘学习 |

## Philosophy

顺势而为 · 敬畏市场 · 严格止损
