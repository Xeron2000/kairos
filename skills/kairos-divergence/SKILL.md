# 分歧理论 - Bit浪浪分歧理论

## 核心概念

分歧理论将主升浪中的调整分为两种：
- **小分歧**：主升浪初期/中期的横盘回踩
- **大分歧**：主升浪中后期的深度调整

## 使用场景

当需要判断调整类型和介入时机时使用此skill：
- 识别调整是小分歧还是大分歧
- 寻找介入机会
- 制定交易策略

## CLI命令

```
MCP tool: detect_signal
参数: symbol (string)

分歧判断主要依靠 LLM 对价格结构的推理，而非独立 CLI 工具。
Hermes 调用 detect_signal 获取基础信号后，结合本 skill 的分歧理论进行深度判断。
```

## 分歧类型

### 小分歧（主升浪初期/中期）
**特征：**
- 行情刚突破新高后第一次整理
- 调整幅度较小（通常5-15%）
- 时间较短（1-5天）
- 成交量萎缩

**形态：**
- 横盘箱体整理
- 不跌破关键支撑
- 结构收敛

**介入策略：**
- 放弃打突破
- 在箱体底部低点做承接
- 止损放在箱体下沿

### 大分歧（主升浪中后期）
**特征：**
- 行情经历连续拉升后深度调整
- 调整幅度较大（15-30%）
- 时间较长（5-15天）
- 可能跌破短期均线

**形态：**
- 快速下跌后企稳
- 可能形成双底结构
- 结构重新收敛

**介入策略：**
- 耐心等待调整结束
- 盘整结构收敛好
- 出现企稳向上迹象时介入
- 博弈主升浪的"二波"行情

## 分歧识别算法

### 小分歧识别
1. **条件**：
   - 主升浪初期/中期
   - 突破新高后第一次整理
   - 调整幅度 < 15%
   - 时间 < 5天

2. **形态**：
   - 横盘箱体
   - 不跌破前低
   - 成交量萎缩

### 大分歧识别
1. **条件**：
   - 主升浪中后期
   - 连续拉升后调整
   - 调整幅度 > 15%
   - 时间 > 5天

2. **形态**：
   - 快速下跌
   - 可能跌破短期均线
   - 企稳后收敛

## 交易策略

### 小分歧策略
- **进场**：箱体底部低点做承接
- **止损**：箱体下沿
- **目标**：前高或更高
- **仓位**：正常仓位

### 大分歧策略
- **进场**：调整结束，结构收敛后
- **止损**：调整低点
- **目标**：二波行情
- **仓位**：轻仓试探

## LLM判断流程

hermes agent使用此skill时：
1. 调用 MCP tool `detect_signal` 获取基础信号
2. 结合本 skill 的分歧理论（小分歧/大分歧）深度分析价格结构
3. 评估调整幅度和时间
4. 寻找介入机会
5. 制定交易计划

## 注意事项

- 小分歧是最佳介入机会
- 大分歧需要耐心等待
- 严格止损是关键
- 结合大盘周期判断

## 实战案例

### BTC/USDT 小分歧案例
- **背景**：主升浪初期，突破新高后第一次整理
- **调整**：从68,500调整到65,000（-5.1%）
- **时间**：3天
- **形态**：横盘箱体整理
- **介入**：65,000做承接
- **止损**：64,200（箱体下沿）
- **目标**：68,500（前高）
- **结果**：成功到达目标，盈利5.3%

### ETH/USDT 大分歧案例
- **背景**：主升浪中后期，连续拉升后深度调整
- **调整**：从3,500调整到3,000（-14.3%）
- **时间**：8天
- **形态**：快速下跌后企稳
- **介入**：3,050企稳后介入
- **止损**：2,950（调整低点）
- **目标**：3,500（前高）
- **结果**：成功到达目标，盈利14.8%
