# 新币交易 - Bit浪浪新币交易窍门

## 核心概念

新币交易窍门：
- **新币上市初期：坚决不做多**
- **寻找结构"做空"**
- **等待深度洗盘后再做多**

## 使用场景

当遇到新上市币种时使用此skill：
- 判断新币是否可交易
- 寻找做空机会
- 等待做多时机

## CLI命令

```
MCP tool: scan_symbols
参数: 无

新币筛选目前由 Hermes 根据上市时间、价格结构等信息进行 LLM 判断，

```

## 新币特征

### 上市初期（0-30天）
- 热度极高，散户关注度高
- 一级市场持仓成本极低
- 抛压极其沉重
- 走势极不稳定

### 结构特征
- 上市后快速冲高
- 随后开始下跌
- 形成高位箱体或下降通道
- 成交量逐渐萎缩

## 交易策略

### 做空策略（上市初期）
**条件：**
- 上市不久（0-30天）
- 初期横盘收敛出看似要突破的结构
- 抛压沉重

**进场：**
- 高位收敛后向下破位时做空
- 或反弹到阻力位做空

**止损：**
- 收敛区间上沿
- 或反弹高点

**目标：**
- 上市低点下方
- 深度洗盘位置

### 做多策略（深度洗盘后）
**条件：**
- 经历极其深度的下跌
- 将早期获利盘彻底洗干净
- 在底部进行长时间盘整
- 重新企稳向上突破

**进场：**
- 底部盘整后向上突破时
- 确认洗盘完成

**止损：**
- 底部盘整区间下沿

**目标：**
- 新币爆发展现出高倍数涨幅

## 风险控制

### 新币特殊风险
1. **流动性风险**：深度差，滑点大
2. **波动性风险**：价格波动剧烈
3. **信息不对称**：项目方可能抛售
4. **情绪化交易**：容易被FOMO影响

### 风险控制措施
1. **轻仓试探**：仓位不超过10%
2. **严格止损**：止损必须设置
3. **避免追高**：绝不追涨新币
4. **耐心等待**：等待明确信号

## LLM判断流程

hermes agent使用此skill时：
1. 扫描新上市币种
2. 分析上市时间和结构
3. 判断是做空还是等待做多
4. 制定交易计划
5. 严格风险控制

## 注意事项

- 新币上市初期坚决不做多
- 做空是主要策略
- 等待深度洗盘后再考虑做多
- 严格止损，轻仓试探

## 实战案例

### 新币做空案例：XXX/USDT
- **上市时间**：10天
- **上市高点**：$16.80
- **当前价格**：$14.50
- **结构**：高位收敛，看似要突破
- **做空信号**：向下破位$14.00
- **进场**：$14.00做空
- **止损**：$15.00（收敛上沿）
- **目标**：$11.00（上市低点下方）
- **结果**：成功到达目标，盈利21.4%

### 新币做多案例：YYY/USDT
- **上市时间**：90天
- **上市高点**：$25.00
- **深度洗盘**：下跌到$5.00（-80%）
- **底部盘整**：60天
- **做多信号**：向上突破$6.00
- **进场**：$6.00做多
- **止损**：$5.00（盘整下沿）
- **目标**：$15.00（前期高点）
- **结果**：成功到达目标，盈利150%
