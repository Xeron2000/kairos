# 选币扫描 - kairos选币策略

## 核心概念

kairos选币公式：
```
大盘拐点企稳 + 属于次新/热点龙头 + 日线结构完美且上方无压力 + 底层箱体蓄力充分
```

## 使用场景

当需要筛选潜在交易标的时使用此skill：
- 寻找龙头币
- 筛选次新币
- 识别热点板块

## 量化筛选条件

### 基础过滤
- **最小成交量**：$80,000,000（24小时）
- **最小持仓量**：$25,000,000
- **最小上市天数**：45天
- **最大波动率**：6.0%

### 优先级排序
1. **次新币**：上市45-180天，上方无套牢盘
2. **热点板块**：AI、动物园系列、Meme
3. **与大盘共振**：与BTC同时启动
4. **抗跌性**：BTC回调时拒绝下跌

## 选币逻辑

### 次新币特征
- 上市不久、自带高人气
- 上方筹码干净，没有历史套牢盘压力
- 一旦资金点火突破，极易拉出大幅度主升浪

### 热点题材
- 每一轮行情都会有资金扎堆的热点
- 自带Meme属性的"动物园"系列
- 群众基础好，牛市中极易吸纳狂热资金

### 与大盘共振
- 真正优质交易币种，启动时机与大盘突破高度吻合
- 当大饼结束调整向上突破时，第一时间寻找与大盘同时启动的山寨币

### 强势抗跌
- 在大饼第一波主升浪拉完后观察山寨币表现
- 那些在大饼震荡回调时极度抗跌的币种，是极佳"种子选手"

## LLM分析流程

hermes agent使用此skill时：
1. 调用 `pwatch scan` 获取候选池
2. 分析每个币种的日线结构
3. 检查上方是否有压力
4. 评估与大盘的相关性
5. 综合判断是否值得交易

## 注意事项

- 量化筛选只是第一步，LLM深度分析更重要
- 需要结合日线结构和上方空间判断
- 与大盘共振是必要条件
- 抗跌性是重要的筛选标准
