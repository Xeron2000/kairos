# 选币完美公式 - Bit浪浪选币策略

## 核心公式

```
大盘拐点企稳 + 属于次新/热点龙头 + 日线结构完美且上方无压力 + 底层箱体蓄力充分
```

## 使用场景

当需要筛选高确定性交易标的时使用此skill：
- 寻找符合所有条件的完美标的
- 评估币种是否值得重仓
- 制定交易计划

## CLI命令

```
MCP tool: scan_symbols
参数: formula="perfect"

返回: 按完美公式评分排序的候选列表
```

## 量化筛选条件

### 1. 大盘拐点企稳（必要条件）
- BTC结束下跌或震荡
- 开始向上突破
- 成交量放大
- 资金费率正常

### 2. 次新/热点龙头（优先条件）
- **次新币**：上市45-180天
- **热点板块**：AI、动物园、Meme
- **龙头特征**：涨幅最大、人气最高

### 3. 日线结构完美（技术条件）
- 上方无密集套牢盘
- 距离强阻力位有足够空间
- 日线级别趋势向上
- 结构清晰，无混乱形态

### 4. 底层箱体蓄力充分（结构条件）
- 经历大级别下跌或长时间震荡
- 构建长时间箱体盘整
- 盘整时间越久越好
- 结构收敛饱满（二次探顶/底）

## 评分系统

### 评分维度（总分100）
1. **大盘共振**（25分）：与BTC启动时机吻合程度
2. **箱体蓄力**（25分）：箱体收敛程度和时间
3. **上方空间**（25分）：距离强阻力位的距离
4. **币种质量**（25分）：次新、热点、龙头特征

### 评分标准
- **90-100分**：完美标的，可重仓
- **80-89分**：优质标的，正常仓位
- **70-79分**：一般标的，轻仓试探
- **<70分**：不符合条件，放弃

## 选币流程

### 第一步：大盘分析
1. 判断市场周期阶段
2. 检查BTC趋势
3. 评估市场情绪

### 第二步：候选池筛选
1. 扫描所有交易对
2. 过滤成交量、OI、上市时间
3. 初步筛选符合基本条件的币种

### 第三步：深度分析
1. 分析日线结构
2. 检查上方压力位
3. 评估箱体蓄力程度
4. 计算与大盘相关性

### 第四步：综合评分
1. 对每个维度打分
2. 计算总分
3. 排序选出最优标的

## LLM分析流程

hermes agent使用此skill时：
1. 调用 MCP tool `scan_symbols(formula="perfect")` 获取候选池
2. 对每个候选进行深度分析
3. 评估是否符合完美公式
4. 制定交易计划（进场、止损、目标）
5. 决定是否执行交易

## 注意事项

- 完美标的很少，耐心等待
- 不符合公式宁可不做
- 大盘是必要条件
- 箱体蓄力是关键
